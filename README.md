
# Kaomoji Status

Fun, animated kaomojis in your VS Code status bar! (≧◡≦)

## Features
- Random or category-based kaomoji in the status bar
- Click to change the kaomoji (with animation)
- Custom kaomojis and interval settings
- Special kaomojis on holidays and your birthday

## Usage
1. Open VS Code settings and search for "Kaomoji Status"
2. Set your preferred category, custom kaomojis, interval, or birthday
3. Click the status bar kaomoji or use the command palette to change or copy it

**Example settings:**
```json
{
  "kaomojiStatus.customKaomojis": ["(≧◡≦)", "(✿◠‿◠)"],
  "kaomojiStatus.interval": 10,
  "kaomojiStatus.category": "cute"
}
```

Enjoy! Holiday and birthday kaomojis appear automatically on special days. (づ￣ 3￣)づ
